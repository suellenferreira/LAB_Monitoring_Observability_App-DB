# Support

This project is an **educational/demo** environment for Azure monitoring and observability. It is maintained on a best-effort basis.

## Getting Help

| Channel | Use For |
|---------|---------|
| [GitHub Issues](https://github.com/suellenferreira/LAB_Monitoring_Observability_App-DB/issues) | Bug reports, feature requests, deployment questions |
| [README](README.md) | Setup instructions, architecture, deployment options |
| [CONTRIBUTING](CONTRIBUTING.md) | How to contribute, local validation steps |

## Before Opening an Issue

1. Check the [README troubleshooting](#) and existing [Issues](https://github.com/suellenferreira/LAB_Monitoring_Observability_App-DB/issues?q=is%3Aissue) for similar problems
2. Include your `deploy-config.cfg` (redacted), the Azure region, and the full error output
3. Specify whether you deployed via pipeline (git push / workflow dispatch) or local CLI

## Azure Product Support

For issues with Azure services themselves (not this demo), use:

- [Azure documentation](https://learn.microsoft.com/azure/)
- [Azure support plans](https://azure.microsoft.com/support/plans/)
- [Microsoft Q&A](https://learn.microsoft.com/answers/)

## No SLA

This is a personal educational project. There is no guaranteed response time or support commitment.
